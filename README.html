TreeViz - ASCII Tree Visualization Tool

Overview

TreeViz is a lightweight, browser-based tool that converts indented text into visually appealing ASCII tree diagrams. It's designed for developers, technical writers, and anyone who needs to visualize hierarchical structures.

Features

  • Real-time conversion
  • Local storage persistence
  • Markdown compatibility
  • Clean, modern interface
  • No installation required

Installation

  1. Clone the repository: git clone https://github.com/yourusername/TreeViz.git
  2. Open index.html in your web browser

Requirements

  • Any modern web browser
  • No additional dependencies
