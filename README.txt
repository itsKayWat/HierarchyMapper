TreeViz - ASCII Tree Visualization Tool
=====================================

A simple, elegant solution for converting plain text hierarchies into ASCII tree diagrams.

OVERVIEW
--------
TreeViz transforms indented text using '=' symbols into visually appealing tree structures. Perfect for documentation, project planning, and hierarchy visualization.

INSTALLATION
-----------
1. Clone the repository
2. Open index.html in any modern web browser
3. Start creating trees!

REQUIREMENTS
-----------
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No additional software needed

USAGE
-----
1. Enter text in the left panel using '=' for indentation levels
2. Watch as your tree generates automatically on the right
3. Copy the generated tree for use in your documentation

Example Input:
project
=src
==components
=docs

Example Output:
├── project
│   ├── src
│   │   └── components
│   └── docs

USE CASES
---------
- Software project documentation
- File system visualization
- Organization charts
- Content hierarchies
- Documentation structures
- Website sitemaps

The tool automatically saves your work locally, so you won't lose your progress when closing the browser.
